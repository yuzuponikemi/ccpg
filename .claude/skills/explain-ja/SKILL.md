# 日本語コード解説スキル

`$ARGUMENTS` で指定されたファイルのコードを日本語で詳しく解説してください。

## 解説フォーマット

```
## コード解説: <ファイル名>

### 概要
このモジュールは〜を目的としています。

### 構成要素
1. **<関数名/クラス名>**: 〜の処理を行う
   - 引数: ...
   - 戻り値: ...
   - 処理の流れ: ...

### 依存関係
- ...

### 改善の余地
- ...
```

## 注意事項
- 初心者にも分かるよう、専門用語には補足説明を加える
- コードの意図（なぜそう書いたか）にも言及する
- 型ヒントの意味も解説に含める
